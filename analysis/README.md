# Analysis

R and Python scripts for data ingestion, transformation, and analysis. Organized by jurisdiction and topic.

## Structure

```
analysis/
├── global/        # WHO API queries, GBD analysis
├── usa/           # CDC WONDER queries, federal spending analysis
├── four-corners/  # State Socrata API queries, state comparisons
├── navajo-nation/ # PDF extraction, NEC report processing
├── south-africa/  # Stats SA PDF extraction, Vulekamali budget analysis
└── cross-cutting/ # Multi-jurisdictional comparisons
```
