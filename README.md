# Public Health Analytics Playbook

**Exploring Public Health Issues with Reproducible Code**

An [Intersect Collaborations LLC](https://www.intersectcollaborations.com/) project.

## About

The Public Health Analytics Playbook is a collection of data-driven analyses exploring real public health issues. Each chapter focuses on a specific topic, pulls data from public sources, and provides reproducible R and Python code readers can adapt for their own work.

## Built With

- [Quarto](https://quarto.org/) (book format)
- [R](https://www.r-project.org/) and [Python](https://www.python.org/) for analysis
- [GitHub Pages](https://pages.github.com/) for publishing
